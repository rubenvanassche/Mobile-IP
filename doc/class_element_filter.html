Detailed Description

Base class for filter predicates for elements and ports.

Deprecated:
This class is deprecated. Use RouterVisitor instead.

ElementFilter objects are used to search the router configuration graph for matching elements. They are usually passed to the Router::downstream_elements() and Router::upstream_elements() functions.

Determine whether an element or port matches this filter.

Parameters
eelement
isoutputtrue for output ports, false for input ports
portport number, or -1 to check the element as a whole

This virtual function is the core of ElementFilter's functionality. The function should return true iff the specified element or port matches the filter. isoutput and port define the interesting port; if port < 0, the element should be checked as a whole.

The default implementation returns false for any element.
