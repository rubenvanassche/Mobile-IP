Detailed Description

template<typename T>
class TokenBucketJiffyParameters< T >

Helper class for token bucket rate limiter.

never Pass this class as the parameter to TokenRateX. TokenBucketJiffyParameters measures ticks in units of jiffies. The template parameter is the type of tokens.

inlinestatic

Return b - a, assuming that b was measured after a.

Some time measurements can, in rare cases, appear to jump backwards, as timestamps do when the user changes the current time. If this happens, and b < a (even though b happened after a), time_monotonic_difference must return 0.
