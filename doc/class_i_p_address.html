Detailed Description

An IPv4 address.

The IPAddress type represents an IPv4 address. It supports bitwise operations like & and | and provides methods for unparsing IP addresses into ASCII dotted-quad form.

Parameters
maskprefix mask

Longer prefix masks are more specific than shorter ones. For example, make_prefix(20).mask_as_specific(make_prefix(18)) is true, but make_prefix(10).mask_as_specific(make_prefix(14)) is false.

Equivalent to (*this & mask) == mask.

Both this address and mask must be prefix masks – i.e., mask_to_prefix_len() returns 0-32. Returns true iff this address contains a longer prefix than mask. For example, make_prefix(20).mask_more_specific(make_prefix(18)) is true, but make_prefix(20).mask_more_specific(make_prefix(20)) is false.

◆ unparse_mask()

String IPAddress::unparse_mask ( ) const

Unparses this address into IP address mask format: either a prefix length or unparse().

If mask_to_prefix_len() >= 0, then returns that value as a base-10 String; otherwise, returns unparse(). Example results include "8" (for 255.0.0.0) and "18.26.4.9".
