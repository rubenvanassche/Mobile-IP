Detailed Description

Member of an 'ar' archive.

The ArchiveElement class represents members of ar(1) archives. Click uses the simple ar(1) format for passing configurations with additional information, such as compiled packages and element maps. An archive member consists of a name, data, and additional metadata. Complete archives are represented as vectors of ArchiveElement objects.
